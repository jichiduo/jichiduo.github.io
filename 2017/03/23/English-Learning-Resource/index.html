English Learning Resource

发表于 2017-03-23 | 分类于 English | | 阅读次数

说明

以下内容为我自己收集的认为对学习英语有用的网站和应用的介绍，每个内容都有相应的点评，希望这些内容可以在你学习英语的路上给你带来一些帮助。

Website - 各种网站类

  1. Book reading suggestion - 专门给儿童介绍适合阅读的书目的网站，同样适合英文初学者

    http://www.arbookfind.com/

  2. The Oxford 3000 - 别的不用说了，赶紧把这3000个词记心里吧 Must try

    http://www.oxfordlearnersdictionaries.com/wordlist/english/oxford3000/

  3. Italki - 你想学英语，好多老外也想学汉语，来这里一帮一，一对红吧

    https://www.italki.com

  4. Quora - 对，中国的知乎就是抄袭的这个网站。 问答类网站，你可以在这里找到很多英语学习的内容

    https://www.quora.com/

  5. Hack News - IT业界的教父级人物Paul Graham的HN, 你如果是搞IT的应该知道这个网站，其实这里更多的是科技类的内容

    https://news.ycombinator.com/news

  6. Buzzfeed - 美国最火的新闻类网站，去看看吧，有趣的内容挺多

    https://www.buzzfeed.com/

  7. TED - 大名鼎鼎的TED,最幸福的是这里的视频都带原版的英文字幕 Ideas worth spreading

    http://www.ted.com/

App - 各种应用类

  1. 扇贝 - 扇贝是中国最大的移动互联网英语学习平台，把全家桶下载下来吧，都不错

    https://www.shanbay.com/

  2. Voscreen - 各种视频短片，各种问题，寓教于乐，值得推荐

    https://www.voscreen.com/

  3. Memrise - 帮你背单词的软件，可以试试

    https://www.memrise.com/
