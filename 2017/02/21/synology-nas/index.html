群晖DS216 Play更换硬盘

发表于 2017-02-21 | 分类于 随笔 | | 阅读次数

缘由

老板自己购买的双盘位的群晖的DS216 Play出了些问题，两块盘中的一块可能出现些问题，在日志中频繁出现了同步错误，并且开始出现Bad sector的错误。
老板自己询问了售后，对方说直接把坏的盘拿下来换块新的盘上去就可以实现自动同步，但是新硬盘挂到系统上以后，系统报已损毁的错误，仔细查询后发现数据都还在，可以使用共享文件夹的方式将数据拷贝出来。

要点记录

  1. 遇到数据问题，第一时间告诉用户不要着急，也许一个错误的决定会给后面的数据恢复带来更大的困难
  2. 群晖DSM系统是基于UBUNTU系统的一个修改版本
  3. 群晖的DSM系统会在硬盘中划分一部分区域并把他的系统文件安装到这个区域中，而且每一块盘都有这个系统，只要你还有一块可以正常工作的盘在，你的DSM应该是可以运行起来的
  4. 如果DSM无法引导的时候，你需要一台PC并把NAS硬盘挂在这个PC上，用U盘版的ubuntu启动系统，然后在系统里把NAS硬盘里的数据拷贝到其他地方
  5. 只要你的硬盘不是硬件损坏，你的数据丢的可能行就很小。数据恢复就是胆大心细的事情，胆大是操作的前提但是更重要的是要自己脑子里面清楚，弄明白怎么去做
  6. 遇到你拿不准的时候，用skype 打个电话给Synology的客服电话：+886 2 2552 5900，售后工程师会耐心的帮助你解决你的问题，并给你最权威的技术指导。
